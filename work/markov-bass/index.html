Markov Bass Generator

Algorithmic music generation using probabilistic models

A generative music tool that creates bass lines using Markov chains. The system analyzes note transitions and produces new sequences that retain stylistic coherence while introducing variation.

Status: v1 finished

Motive

Musical patterns often emerge from repetition and variation. This project explores how probabilistic models can capture these patterns and generate new material that feels structured but not deterministic.

The goal is to create a simple system capable of producing musically meaningful sequences that can be used as a foundation for composition or live experimentation.

System

The generator is implemented in Python and builds transition matrices from input sequences to model note-to-note probabilities.

  • Markov chain model for sequence generation
  • Transition matrix built from MIDI or predefined patterns
  • Randomized sequence generation with controlled probabilities
  • Output adaptable to MIDI or audio environments

Output

The system produces MIDI bass lines that maintain stylistic continuity while introducing variation, making it useful for sketching ideas or generating evolving musical patterns. At the moment is being used with Pure Data as a sound engine.
