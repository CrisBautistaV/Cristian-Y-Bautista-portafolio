Industrial Monitoring System

Data acquisition and visualization for industrial processes

A monitoring system designed to collect, visualize, and store data from industrial equipment. The system integrates embedded an industrial hardware, communication protocols, and cloud-based tools to provide continuous insight into process variables.

Status: finished — preventive mantenance

Motive

Industrial processes often rely on fragmented or inaccessible data, making monitoring and analysis difficult. This project was developed to create a simple and reliable bridge between physical systems and accessible digital tools.

The goal was to design a system that could be easily deployed, understood, and adapted, enabling real-time visibility without introducing unnecessary complexity, considering client budget and timeframe, allowing scaling in the future.

System

The system connects industrial equipment to a Raspberry Pi, which handles data acquisition, processing, and communication.

  • Raspberry Pi as central node
  • Communication with PLCs via Modbus TCP
  • Data acquisition from multiple variables (food industry)
  • Integration with Google Sheets for cloud storage
  • Desktop interface built with Qt for visualization

Output

The system provides continuous logging of process variables, remote access to data, and a simple interface for monitoring system behavior.

It has been used to track industrial oven performance, enabling better visibility and supporting decision-making in operation and maintenance.
